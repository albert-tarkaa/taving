## Running Locally

This application requires Node.js v16.13+.

```bash
git clone https://github.com/albert-tarkaa/taving.git
cd taving
npm install
npm run dev 
```

Create a `.env` file similar to [`.env.example`](https://github.com/albert-tarkaa/taving/main/.env.example).
